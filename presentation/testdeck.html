# Type Safe Image Hosting in [Haskell](http://haskell.org) and [Yesod](http://www.yesodweb.com/)
## Why [Yesod](http://www.yesodweb.com/) ? 1. **Secure and Type Safe** * Leverages all the strengths of [Haskell](http://haskell.org)'s strong type system like **type safe urls** . * Protects against a lot of security threats . 2. **RESTful MVC framework** * Database interaction with **[Persistent](http://www.yesodweb.com/book/persistent)** . 3. **Fast and Scalable** * Uses **[warp](http://www.yesodweb.com/blog/2011/02/warp-speed-ahead)** , a very fast web server written in [Haskell](http://haskell.org). * Other fast libraries like **[blaze-builder](http://hackage.haskell.org/package/blaze-builder)** to efficiently generate html , **[aeson](https://github.com/mailrank/aeson)** to handle json . 4. **Productive** * Increased productivity using advanced [Haskell](http://haskell.org) features like **template Haskell** and **quasi-quoting**.
